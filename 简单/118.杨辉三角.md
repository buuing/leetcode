
## 118.杨辉三角 [简单]

<br />

> 力扣（LeetCode）https://leetcode-cn.com/problems/pascals-triangle

<br />

给定一个非负整数 numRows，生成杨辉三角的前 numRows 行。

在杨辉三角中，每个数是它左上方和右上方的数的和。

**示例:**

```
输入: 5
输出:
[
     [1],
    [1,1],
   [1,2,1],
  [1,3,3,1],
 [1,4,6,4,1]
]
```

---

> 通过15个测试用例, 执行用时：64 ms

```js
const generate = n => {
  if (n === 0) return []
  if (n === 1) return [[1]]
  const arr = [[1]]
  for (let i = 1; i < n; i++) {
    arr[i] = [1]
    for (let j = 1; j <= i; j++) {
      if (j !== i) {
        arr[i][j] = arr[i - 1][j - 1] + arr[i - 1][j]
      } else {
        arr[i][j] = 1
      }
    }
  }
  return arr
}
```
