
## 328.奇偶链表 [中等]

<br />

> 力扣（LeetCode）https://leetcode-cn.com/problems/odd-even-linked-list

<br />

给定一个单链表，把所有的奇数节点和偶数节点分别排在一起。请注意，这里的奇数节点和偶数节点指的是节点编号的奇偶性，而不是节点的值的奇偶性。

请尝试使用原地算法完成。你的算法的空间复杂度应为 O(1)，时间复杂度应为 O(nodes)，nodes 为节点总数。

**示例 1:**

```
输入: 1->2->3->4->5->NULL
输出: 1->3->5->2->4->NULL
```

**示例 2:**

```
输入: 2->1->3->5->6->4->7->NULL 
输出: 2->3->6->7->1->5->4->NULL
```

**说明:**

- 应当保持奇数节点和偶数节点的相对顺序。
- 链表的第一个节点视为奇数节点，第二个节点视为偶数节点，以此类推。

---

> 通过71个测试用例, 执行用时：164 ms

```js
// 暴力解题
const oddEvenList = head => {
  let curr = head, arr = [], res = null
  let flag = true
  while (curr) {
    arr.unshift(curr.val)
    if (!curr.next || !curr.next.next) {
      curr = flag ? head.next : null
      flag = false
    } else {
      curr = curr.next.next
    }
  }
  for (let i = 0; i < arr.length; i++) {
    res = { val: arr[i], next: res }
  }
  return res
}
```

> 通过71个测试用例, 执行用时：128 ms

```js
// 双指针原地算法
const oddEvenList = head => {
  if (!head || !head.next || !head.next.next) return head
  let odd = head, even = head.next
  let temp = even
  while (odd && even && even.next) {
    odd.next = even.next
    odd = odd.next
    even.next = odd.next
    even = even.next
  }
  odd.next = temp
  return head
}
```
